What is ANTLR

Generates LL(*) parser. It takes grammar on the input and generates a parser in a target language (originally Java, also python, go, js, ts) on the output, also provides several helpers to walk the parse tree.

How parser generated by ANTLR works

ANTLR Grammar

expr: expr ( '*' | '/' ) expr
    | expr ( '+' | '-' ) expr
    | INT
    | ID
    | '(' expr ')‚Äô
    ;

ID  : [A-Za-z_][A-Za-z_0-9]* ; // match identifiers
INT : [0-9]+ ;                 // match integers
NL  : '\r'? '\n' ;             // newlines
WS  : [ \t\r\n\f]+ -> skip ;   // skip whitespace

Parse Tree for

a + b * (var3 / 2)

Styleguide Control

DATA CLASS CLASS18;
SET SASHELP.CLASS;
BMI=WEIGHT/HEIGHT**2;
IF AGE > 18 THEN DO; OUTPUT CLASS18; END;
ELSE OUTPUT CLASS;
PROC FREQ DATA = CLASS18;
TABLES BMI;
RUN;
data class class18;
  set sashelp.class;
  bmi = weight / height ** 2;
  if age > 18 then do;
    output class18;
  end;
  else do;
    output class;
  end;
run;

proc freq data = class18;
  tables bmi;
run;
